# Usage Guide

## Prerequisites
- Android Studio (Hedgehog or newer recommended).
- An OpenWeather API key.

## Configure the API key
Replace the value of `OPEN_WEATHER_API_KEY` in `app/build.gradle` with your OpenWeather API key.

## Run the app
1. Open the project in Android Studio.
2. Sync Gradle.
3. Select a device/emulator and run the app.

## Tips
- Use a valid city name (e.g., "Warsaw" or "San Francisco").
- If a request fails, the app will show an error message; try again with a different city.
